## JavaScript Fundamentals: The Building Blocks

### 1. Closures & Scope

Closures are one of the most commonly asked interview topics. Here's a practical example:

```javascript
function createCounter() {
  let count = 0;
  return {
    increment: () => ++count,
    getCount: () => count
  };
}

const counter = createCounter();
console.log(counter.increment()); // 1
console.log(counter.getCount()); // 1
```

This demonstrates how closures maintain access to their outer scope even after the outer function has returned.

### 2. Promises & Async Programming

Modern JavaScript heavily relies on asynchronous programming. Here's how to handle async operations effectively:

```javascript
async function fetchUserData() {
  try {
    const response = await fetch('https://api.example.com/user');
    const data = await response.json();
    return data;
  } catch (error) {
    console.error('Error fetching user:', error);
    throw error;
  }
}
```

### 3. The Event Loop

Understanding the event loop is crucial for performance optimization:

```javascript
console.log('1');
setTimeout(() => console.log('2'), 0);
Promise.resolve().then(() => console.log('3'));
console.log('4');

// Output: 1, 4, 3, 2
```

## React Concepts: The Framework Fundamentals

### 1. Custom Hooks

Let's look at a custom hook that combines several React hooks:

```jsx
function useDebounce(value, delay) {
  const [debouncedValue, setDebouncedValue] = useState(value);

  useEffect(() => {
    const timer = setTimeout(() => {
      setDebouncedValue(value);
    }, delay);

    return () => clearTimeout(timer);
  }, [value, delay]);

  return debouncedValue;
}
```

### 2. Performance Optimization

React's performance optimization techniques are crucial for scaling applications:

```jsx
const MemoizedComponent = React.memo(function ExpensiveComponent({ data }) {
  return <div>{/* Rendered content */}</div>;
});

function ParentComponent() {
  const [count, setCount] = useState(0);
  const expensiveValue = useMemo(() => computeExpensiveValue(count), [count]);
  
  return <MemoizedComponent data={expensiveValue} />;

```

### 3. State Management

Understanding different state management approaches is essential:

```jsx
function reducer(state, action) {
  switch (action.type) {
    case 'increment':
      return { count: state.count + 1 };
    case 'decrement':
      return { count: state.count - 1 };
    default:
      throw new Error();
  }
}
```
