JavaScript Fundamentals: The Building Blocks

1. Closures & Scope

Closures are one of the most commonly asked interview topics. Here‚Äôs a practical example:

function createCounter() {
  let count = 0;
  return {
    increment: () => ++count,
    getCount: () => count
  };
}

const counter = createCounter();
console.log(counter.increment()); // 1
console.log(counter.getCount()); // 1

This demonstrates how closures maintain access to their outer scope even after the outer function has returned.

React Concepts: The Framework Fundamentals

1. Custom Hooks

Let‚Äôs look at a custom hook that combines several React hooks:

function useDebounce(value, delay) {
  const [debouncedValue, setDebouncedValue] = useState(value);

  useEffect(() => {
    const timer = setTimeout(() => {
      setDebouncedValue(value);
    }, delay);

    return () => clearTimeout(timer);
  }, [value, delay]);

  return debouncedValue;
}
